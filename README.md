## About Me:

I am a student at the University of Alberta, currently studying towards my BSc. Spec. in Computing Science.

My interests include GPU programming, low level systems and computer architecture, and deep learning.

I want to learn more about graphics, compilers, and operating systems.

I love the work that I do and the people I work with!

## Previous Projects:
**QrazyQRsRUs**: An Android application for organizing events - with QR codes! Made for CMPUT301 - Introduction to Software Engineering.

**pyPrettyBackground**: An image recommendation program built to recommend scenic desktop backgrounds, similar to your favorites.

**Mind Pilot**: A hands-free computer control program for the natHACKS 2024 hackathon.
